# ClimaFlow - Application Météo

Une application mobile Flutter connectée à une API FastAPI, affichant la météo en temps réel avec une interface moderne.

## 🚀 Fonctionnalités

- **Météo actuelle** détaillée (température, ressenti, humidité, vent, pression)
- **Icônes météo** officielles OpenWeatherMap
- **Recherche de ville** par nom
- **Rafraîchissement manuel**
- **Design moderne** et responsive (Mode sombre inclus)
- **Architecture propre** (Feature-First + Riverpod)
- **Backend FastAPI** complet
- **Dockerisation** complète

## 🛠 Stack Technique

### Backend
- **FastAPI** (Python 3.12)
- **OpenWeatherMap API**
- **SQLAlchemy + AsyncPG** (PostgreSQL prêt)
- **Docker + Docker Compose**

### Frontend
- **Flutter** (Dart)
- **Riverpod** (State Management)
- **Dio** (HTTP Client)
- **GoRouter** (Navigation)
- **Cached Network Image**
